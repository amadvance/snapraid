SnapRAID Daemon (Automation, Web UI, REST API)

Warning:The SnapRAID Daemon for Linux requires SnapRAID CLI 14.0+ to be already installed and configured. In contrast, the Windows installer contains both the SnapRAID CLI and the SnapRAID Daemon. On Windows, if you are already using SnapRAID, ensure you install the daemon in the same directory.
Warning:SnapRAID Daemon is currently available only for Linux and Windows. If you are a Mac developer interested in volunteering your expertise to bring the daemon to macOS as an open-source collaborator, we would love to hear from you!

Includes background service and Web UI. See the ChangeLog for the detailed list of changes, and CheckSums for the SHA256 checksums.
